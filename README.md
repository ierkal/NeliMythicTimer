NeliMythicTimer

Customized M+ Timer with additional features.

Features

Customized Mythic+ Timer.
* Shows time from start counting upwards and time left for +1/+2/+3
* Shows enemies percentage and absolute numbers.
* Shows enemy forces of the current pull. (Removed in Midnight)
* Shows the completion time and stays in UI.
* Death Counter with names and time lost. (Removed in Midnight)
* Shows enemy forces percent in tooltip.   -- Requires Mythic Dungeon Tools. (Removed in Midnight)
* Shows enemy forces percent in nameplates. -- Requires Mythic Dungeon Tools. (Removed in Midnight)
* Automatically inserts the keystone.
* Show affixes as text.
* Demo availabile when outside of dungeon.
* Hides Blizzard UI.

Commands

* /nmt config - Opens the config.
* /nmt demo - Opens Demo Timer
